Working with Formulas

Master the building blocks of Excel to analyse data faster.

Duration · 15 mins

📹 Video: Excel Formulas Basics

Guided walkthrough on creating formulas, using functions, and working with references.

Lesson content

In this lesson you'll learn how to create and use basic formulas in Excel. Formulas are the backbone of Excel's power, allowing you to perform calculations, analyse data, and automate repetitive tasks.

Key topics
  • Understanding cell references
  • Basic arithmetic operations
  • Using functions like SUM, AVERAGE, COUNT
  • Relative vs absolute references
💡 Pro tip: Always start formulas with an equals sign (=) and use parentheses to control the order of operations.
